Tentang Proyek

Proyek ini merupakan pengembangan website Learning Management System (LMS), yaitu sebuah platform pembelajaran berbasis kursus dengan nama "Uni-Learn". Proyek ini saya kerjakan sebagai bagian dari Capstone Project akhir dalam program MSIB (Magang & Studi Independen Bersertifikat). Pada program Studi Independen tersebut, saya berhasil diterima di perusahaan PT. Educa Sisfomedia Indonesia (GAMELAB.ID) dan berkolaborasi bersama tim yang terdiri dari rekan-rekan peserta SIB lainnya. Proyek ini dibangun menggunakan Laravel Framework dengan dukungan Bootstrap serta basis data MySQL. Dalam pengerjaannya, saya berperan sebagai Front-End Developer.
